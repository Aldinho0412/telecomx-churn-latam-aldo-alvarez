# Diccionario de Datos – Telecom X

Este documento describe las variables incluidas en el dataset de clientes de Telecom X.

## Variables principales

| Campo | Descripción |
|------|-------------|
| customerID | Identificador único del cliente |
| gender | Género del cliente |
| SeniorCitizen | Indica si el cliente es adulto mayor |
| Partner | Tiene pareja |
| Dependents | Tiene dependientes |
| tenure | Meses de permanencia |
| PhoneService | Servicio telefónico |
| InternetService | Tipo de servicio de internet |
| Contract | Tipo de contrato |
| PaymentMethod | Método de pago |
| MonthlyCharges | Cargo mensual |
| TotalCharges | Cargo total |
| Churn | Indica si el cliente canceló el servicio |
